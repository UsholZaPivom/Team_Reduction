\# Team Reduction



Проект: автоматизированное средство формирования списка сокращений документа в Microsoft Word.



\## Назначение



Программа последовательно выполняет 3 этапа:



1\. распознавание текста и выделение словоформ, поддающихся сокращению;

2\. вычленение доступных к сокращению словоформ и уже имеющихся аббревиатур;

3\. определение необходимости ввода новых аббревиатур.



\## Требования



Рекомендуемое окружение:

\- Python 3.12

\- документ в формате `.docx`



\## Файлы проекта



Основные файлы проекта:



\- `main.py` — последовательный запуск всех 3 этапов;

\- `text\_recognition\_candidates\_v3.py` — этап 1;

\- `abbreviation\_extraction\_stage2.py` — этап 2;

\- `abbreviation\_need\_stage3.py` — этап 3;

\- `requirements.txt` — зависимости проекта;

\- `pyproject.toml` — описание проекта и зависимостей;

\- `.gitignore` — исключения для Git.
