Инструкция по использованию:

Файл test_reduction_input.docx является тестовыми входными данными (при желании можете его изменить)

Для запуска нужно:

создать окружение на Python 3.12;
выполнить pip install -r requirements.txt;
положить файл test_reduction_input.docx рядом с main.py;
запустить python main.py.

Все результаты автоматически сохраняются в папку result_all по трём этапам обработки.
